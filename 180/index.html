bootmapper client 업데이트

October 4, 2014

bootmapper client를 V0.4.4로 업데이트 했습니다.

기존의 사용자들은 core 파일만 다운로드 하여 기존 폴더에 덮어 씌우면 됩니다.

추가된 기능은 아래와 같습니다.

- 텍스트를 매크로로 변환하는 기능을 추가

예를 들어 “abcdEFG"는 텍스트를 타이핑하고 “parse string »” 버튼을 누르면 매크로에 적용 됩니다.

더 자세한 설명 : http://blog.winkeyless.kr/125

- 해상도가 작은 모니터에서 이용 시 화면을 스크롤 할 수 있도록 수정
