키 입력을 위해서 다이오드는 필수 입니다.

저항은 full LED를 이용할 경우 필요합니다.

SMD 다이오드(1N4148W)

SMD 다이오드(LL4148)

- 두 다이오드 간 성능 차이는 없으며, 자신이 납땜하기에 편한 모양으로 선택하면 됩니다.

2012 칩 저항

820옴 : 보통 흰색/핑크 컬러 LED에 사용

910옴 : 보통 파란색/녹색 LED에 사용

1300옴 : 보통 빨간색/노란색/주황 LED에 사용

3파이 LED나 사각LED(일명 메가톤바)에 비해 모따기 등의 가공이 필요없이 납땜만 하면 사용할 수 있는 최신 트랜드의 LED입니다.
