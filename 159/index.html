B 시리즈 기판 신청

June 25, 2014

B 시리즈 기판들은 컬러를 자유롭게 변환할 수 있는 특수한 RGB LED를 사용할 수 있는 것이 가장 큰 특징입니다.

또한, PC용 클라이언트(bootMapper Client) 애플리케이션을 이용해서 자유롭게 설정이 가능합니다.

새로운 펌웨어인 ps2avrGB 는 기존의 ps2avrU와 대부분의 기능은 동일하며 여기에 RGB LED 컨트롤 기능을 추가한 것 입니다.

아크릴 하우징도 쉽게 주문/제작 할 수 있습니다.

아크릴 하우징 주문 : http://hanaacryl.com/product/?s_category=150

기판 사양

- 레이아웃 (공통)

윈키/윈키리스 콤보,

캡스락 3000/8000(마제식) 지원

스페이스바 마제식 지원

1.75 우 쉬프트 지원

ISO 배열 지원

- 레이아웃 (B.87 EX)

텐키 분리 가능

텐키의 2X키를 1X * 2개로 대체 가능

- RGB LED 지원 : 대부분의 컬러로 자유롭게 변환 가능(옵션으로 RGB LED 소자 세트 구매)

- Full LED 지원 : 5가지 LED 모드

- PCB : 1.6T (화이트 컬러)

- 모든 소자 뒷면에 배치

- 하우징(보강판)리스를 위한 마운트홀 지원

- mac os 호환

ps2avrGB는 하드웨어 키매핑을 제외한 모든 기능이 ps2avrU와 동일합니다.

또한, bootmapper client를 제공하여 LED 설정을 PC에서 손쉽게 변경할 수 있습니다.

ps2avrGB 기능

- RGB LED 컨트롤

- 간단한 키 맵핑 / 매크로 등록

ps2avrU(펌웨어) 사양

- 인터페이스 : USB / PS/2 콤보 (자동 인식)

- 무한 동시입력 지원 (PS/2, USB연결 시 6+1입력)

- Full LED 지원 : 5가지 LED 모드

- 4 레이어의 키맵 지원

- 각종 멀티미디어 키 지원

- 다양한 특수 기능(퀵 스왑/듀얼 액션 등) 지원

- 매크로 지원 (키 입력간 지연 가능)

- 간단한 키 맵핑 지원

- mac os 호환
